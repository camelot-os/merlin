Writing Merlin Drivers
======================

To help users with writing drivers for their devices, this section describes the expected source
layout and build process for Merlin drivers.

It describes, for various device drivers types, the basic structure that Merlin expects and how
drivers should delivers upper layer interfaces in a portable way.
